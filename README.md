**G**reat **O**bserver **O**rganizes **B**rilliant **E**ntertaining **R**esults

A Discord Bot meant to help with hosting trivia shows with friends.

Main functionality includes a buzzer and scoreboard system. There's Save and Load commands in case the bot crashes, but those are currently WIP.

Made in Discord.js v14.18.0 by JustACoolDude, 2025.
